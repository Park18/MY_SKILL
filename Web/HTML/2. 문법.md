# 1. 제목
## 1.1. 제목(Heading)
HTML은 제목을 표현할 수 있는 다양한 크기의 ```<h>``` 태그를 제공한다.
가장 큰 ```<h1>``` 부터 가장 작은 ```<h6>```까지 다양한 크기로 표현할 수 있다.

```
<h1>제목1의 크기이다.</h1>
<h2>제목2의 크기이다.</h2>
<h3>제목3의 크기이다.</h3>
<h4>제목4의 크기이다.</h4>
<h5>제목5의 크기이다.</h5>
<h6>제목6의 크기이다.</h6>
```
<h1>제목1의 크기이다.</h1>
<h2>제목2의 크기이다.</h2>
<h3>제목3의 크기이다.</h3>
<h4>제목4의 크기이다.</h4>
<h5>제목5의 크기이다.</h5>
<h6>제목6의 크기이다.</h6>

> ```<h>``` 태그 위아래로는 약간의 여백이 삽입된다.

# 2. 단락
## 2.1. 단락(Paragraph)
단락이란 내용상 끊어서 구분할 수 있는 하나하나의 부분을 의미, 문단이라고도 불린다. HTML에서는 ```<p>``` 태그를 이용하여 표현한다.

```
<h1>제목1의 크기이다.</h1>
<h1>제목1의 크기이다.</h1>
<h1>제목1의 크기이다.</h1>
<p>단락 시작부분이다.</p>
```
<h1>제목1의 크기이다.</h1>
<h1>제목1의 크기이다.</h1>
<h1>제목1의 크기이다.</h1>
<p>단락 시작부분이다.</p>

> ```<p> 태그 위아래로는 약간의 여백이 자동으로 삽입된다.

## 2.2. 띄어쓰기와 줄 나누기
HTML 코드에서 띄어쓰기나 줄 나누기를 여러번 하더라도 웹 브라우저를 통해 나타나는 화면에는 전혀 영향을 주지 못한다. 웹 브라우저는 여러 번의 띄어쓰기나 줄 나누기를 오직 하나의 띄어쓰기나 줄로만 인식하기 때문이다.

```
<p>
줄을 나누고 싶어서
이렇게 줄을 나줬다.

과연    그래도    출력이    될까?
</p>
```
<p>
줄을 나누고 싶어서
이렇게 줄을 나줬다.

과연    그래도    출력이    될까?
</p>

위의 예제는 여러번의 띄어쓰기와 줄 나누기를 표현하고자 하는 ```<p>``` 태그를 이용한다. 하지만 ```<p>``` 태그 내에서 작성된 여러번의 띄어쓰기와 줄 나누기는 오직 하나의 띄어쓰기로만 표현된다.

```<br>``` 태그(break line)를 사용하면 새로운 단락을 만들지 않고도 줄을 나눌 수 있다. 이러한 ```<br>``` 태그는 종료 태그가 없는 빈 태그(empty tag)이다.

```
<p>
줄을 나누고 싶어서<br>
이렇게 줄을 나줬다.<br>
<br>
과연    그래도    출력이    될까?
</p>
```
<p>
줄을 나누고 싶어서<br>
이렇게 줄을 나줬다.<br>
<br>
과연    그래도    출력이    될까?
</p>

## 2.3. 텍스트 서식 미리 정하기
HTML 코드에서 작성한 텍스트 서식을 그대로 표현하려면 ```<pre>``` 태그를 사용해야한다. ```<pre>``` 태그(preformatted text) 내에 작성된 텍스트의 모든 띄어쓰기와 줄 나누기는 웹 브라우저에 그대로 표현된다.
```
<pre>
줄을 나누고 싶어서
이렇게 줄을 나줬다.

과연    그래도    출력이    될까?
</pre>
```
<pre>
줄을 나누고 싶어서
이렇게 줄을 나줬다.

과연    그래도    출력이    될까?
</pre>

> ```<pre>``` 태그 내에 작성된 텍스트의 글꼴은 고정폭 글꼴로 자동변환된다.

## 2.4. 수평 가로 구분선
단락을 나루 때나 내용상의 구분을 표현하고자 할 때 수평 가로 구분선을 사용한다. 이렇게 사용되는 수평 가로 구분선을 HTML 코드에서는 ```<hr>``` 태그(horizontal rule)로 간단하게 만들 수 있다.
```
<p>단락 1이다.</p>
<hr>
<p>단락 2이다.</p>
<hr>
<p>단락 3이다.</p>
```
<p>단락 1이다.</p>
<hr>
<p>단락 2이다.</p>
<hr>
<p>단락 3이다.</p>

# 3. 서식

## 3.1. 서식(Formatting)
HTML은 텍스트에 다양한 효과를 주는 여러 태그를 제공한다.

## 3.2. 강조효과

### 3.2.1 굵게 
```<b>``` 태그(bold text)나 ```<strong>``` 태그를 사용한다.

```
<p> <b>"이 부분"</b>은 단순히 글씨가 굵은 부분이다.</p>
<p> <strong>"이 부분"</strong>은 중요해서 굷게 표현된 부분이다.</p>
```
<p><b>"이 부분"</b>은 단순히 글씨가 굵은 부분이다.</p>
<p><strong>"이 부분"</strong>은 중요해서 굷게 표현된 부분이다.</p>

## 3.2.2 기울기체(이탤릭체)
```<i>``` 태그(italic text)나 ```<em>``` 태그(emphasized text)를 사용한다.
```
<p><i>"이 부분"</i>은 단순히 글시가 기울기체인 부분이다.</p>
<p><em>"이 부분"</em>은 중요해서 기울기체로 표현된 부분이다.</p>
```
<p><i>"이 부분"</i>은 단순히 글시가 기울기체인 부분이다.</p>
<p><em>"이 부분"</em>은 중요해서 기울기체로 표현된 부분이다.</p>

> 검색엔진은 ```<strong>``` 태그나 ```<em>``` 태그를 사용하여 강조된 텍스트를 더 중요하게 인식한다.

## 3.3. 텍스트 강조(하이라이팅) 효과
```<mark>``` 태그를 사용한다.

```
<p><mark>" 이부분"</mark>은 하이라이팅 처리한 부분이다.</p>
```
<p><mark>" 이부분"</mark>은 하이라이팅 처리한 부분이다.</p>

## 3.4. 삭제효과
```<del>``` 태그(delete)를 사용한다. 텍스트 중앙에 가로줄을 그어 텍스트를 지운 것과 같은 효과를 나태난다.

```
<p><del>" 이부분"</del>은 하이라이팅 처리한 부분이다.</p>
```
<p><del>" 이부분"</del>은 하이라이팅 처리한 부분이다.</p>

## 3.5. 삽입효과
```<ins>``` 태그(insert)를 사용한다. 텍스트 밑에 가로줄을 그어 빈칸에 텍스트를 삽입한 것과 같은 효과를 나타낸다.

```
<p><ins>" 이부분"</ins>은 하이라이팅 처리한 부분이다.</p>
```
<p><ins>" 이부분"</ins>은 하이라이팅 처리한 부분이다.</p>

## 3.6. 윗첨자와 아래첨자 효과
윗첨자로 ```<sup>``` 태그(superscript), 아래첨자로 ```<sub>``` 태그(subscript)를 사용한다.

```
<p>X<sup>2</sup> + Y<sup>3</sup> = Z</p>
<p>H<sub>2</sub>O는 물을 나타내는 화학식이다.</p>
```
<p>X<sup>2</sup> + Y<sup>3</sup> = Z</p>
<p>H<sub>2</sub>O는 물을 나타내는 화학식이다.</p>

# 4. 인용구

## 4.1. 인용구(Quotation)
HTML에서 인용구를 표현하는 방법으로 두 가지가 있다.
1. 짧은 인용구
2. 블록 인용구

## 4.2. 짧은 인용구
```<q>``` 태그(quotation)을 사용한다. 자동으로 앞뒤에 큰 따옴표가 붙는다.

```
<p>HTML의 정의는 <q>웹 페이지를 만들기 위한 하이퍼텍스트 마크업 언어</q>이다.</q>
```
<p>HTML의 정의는 <q>웹 페이지를 만들기 위한 하이퍼텍스트 마크업 언어</q>이다.</q>

## 4.3. 블록 인용구
```<blockquote>``` 태그(block quatation)를 사용한다. 인용 부분을 별도의 단락으로 구분하여 나타낸다.

```
<p>HTML의 정의</p>
<blockquote>
인터넷 서비스의 하나인 www를 통해 볼 수 있는 문서를 만들 때 사용하는 프로그래밍 언어의 한 종류이다.
</blockquote>
```
<p>HTML의 정의</p>
<blockquote>
인터넷 서비스의 하나인 www를 통해 볼 수 있는 문서를 만들 때 사용하는 프로그래밍 언어의 한 종류이다.
</blockquote>

## 4.4. 축약형 표현
```<abbr>``` 태그(abbreviation)를 사용한다. 태그 위에 아우스를 위치시키면 title 속성에 명시된 용어의 원형이 나타난다.

```
<p>
<strong><abbr title="HyperText Markup Language 5">HTML5</abbr></strong>
란 웹 문서를 제작하는데 쓰이는 프로그래밍 언어인 HTML의 최신규격이다.
</p>
```
<p>
<strong><abbr title="HyperText Markup Language 5">HTML5</abbr></strong>
란 웹 문서를 제작하는데 쓰이는 프로그래밍 언어인 HTML의 최신규격이다.
</p>

## 4.5. 주소표현
```<address>``` 태그를 사용한다. 기울기체로 표현되며, 위아래로 약간의 공백이 자동으로 삽입된다.

```
<address>
충청남도<br>
아산시
</address>
```
<address>
충청남도<br>
아산시
</address>
